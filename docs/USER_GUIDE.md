# PixeLab 사용자 가이드

## 시작하기

### 첫 실행

PixeLab을 처음 실행하면 32x32 픽셀의 빈 캔버스가 나타납니다.

### 인터페이스 구성

```
┌─────────────────────────────────────────────────────┐
│  File  Edit  View  Help                            │
├────┬────────────────────────────────────────┬───────┤
│    │                                        │       │
│ 도구│           캔버스 영역                   │ 색상  │
│    │                                        │ 팔레트│
│    │                                        │       │
│    │                                        │       │
├────┴────────────────────────────────────────┴───────┤
│  상태 표시줄: 커서 위치, 캔버스 크기, 줌 레벨        │
└─────────────────────────────────────────────────────┘
```

## 도구 사용법

### 연필 (Pencil)
- **단축키**: `P`
- **사용법**: 클릭하거나 드래그하여 단일 픽셀 그리기
- **용도**: 정밀한 픽셀 단위 작업

### 브러시 (Brush)
- **단축키**: `B`
- **사용법**: 드래그하여 여러 픽셀 그리기
- **크기 조정**: 도구 옵션에서 브러시 크기 선택 (1-10픽셀)
- **용도**: 빠른 채색 작업

### 지우개 (Eraser)
- **단축키**: `E`
- **사용법**: 드래그하여 픽셀 삭제 (투명하게)
- **크기 조정**: 브러시와 동일하게 크기 조정 가능

### 채우기 (Fill/Bucket)
- **단축키**: `F`
- **사용법**: 클릭한 영역을 현재 색상으로 채우기
- **동작**: 같은 색상으로 연결된 영역 전체를 채움
- **주의**: 대규모 영역은 시간이 걸릴 수 있음

### 스포이드 (Eyedropper)
- **단축키**: `I`
- **사용법**: 캔버스의 픽셀을 클릭하여 색상 추출
- **팁**: 모든 도구 사용 중 `Alt` 키를 누르면 임시 스포이드 모드

### 선 도구 (Line)
- **단축키**: `L`
- **사용법**: 클릭-드래그-릴리스로 직선 그리기
- **팁**: `Shift` 키를 누르면 45도 단위로 스냅

### 사각형 도구 (Rectangle)
- **단축키**: `R`
- **사용법**: 클릭-드래그-릴리스로 사각형 그리기
- **옵션**:
  - 채움 (Filled): 내부를 색상으로 채움
  - 윤곽선 (Outline): 테두리만 그림
- **팁**: `Shift` 키로 정사각형 그리기

### 원 도구 (Circle)
- **단축키**: `C`
- **사용법**: 클릭-드래그-릴리스로 원 그리기
- **옵션**: 사각형과 동일 (채움/윤곽선)
- **팁**: `Shift` 키로 정원 그리기

## 색상 작업

### 색상 선택

1. **팔레트에서 선택**: 우측 팔레트에서 저장된 색상 클릭
2. **색상 선택기**: "색상 선택" 버튼으로 RGB 색상 다이얼로그 열기
3. **스포이드**: 캔버스에서 기존 색상 추출

### 색상 팔레트 관리

- **색상 추가**: "팔레트에 추가" 버튼으로 현재 색상 저장
- **색상 삭제**: 팔레트의 색상을 우클릭하여 삭제
- **기본 팔레트**: 
  - 검정, 흰색, 회색
  - 빨강, 초록, 파랑
  - 노랑, 사이안, 마젠타

### 투명도

- 투명 픽셀은 체크보드 패턴으로 표시됩니다
- PNG로 내보낼 때 투명도가 유지됩니다

## 캔버스 조작

### 줌 (확대/축소)

1. **마우스 휠**: 커서 위치 중심으로 줌
2. **줌 슬라이더**: 상단 바의 슬라이더 조정
3. **줌 버튼**: `+` / `-` 버튼 클릭
4. **단축키**:
   - `Ctrl` + `+`: 확대
   - `Ctrl` + `-`: 축소
   - `Ctrl` + `0`: 100% (실제 크기)

**줌 범위**: 10% ~ 5000%

### 패닝 (이동)

1. **Space + 드래그**: 캔버스를 자유롭게 이동
2. **중클릭 드래그**: 마우스 휠 클릭 후 드래그

### 격자 표시

- **단축키**: `G`
- **메뉴**: View → Show Grid
- 격자는 픽셀 경계를 명확하게 표시합니다
- 높은 줌 레벨에서 자동으로 표시됩니다

## 파일 작업

### 새 캔버스 만들기

1. **File → New** 또는 `Ctrl+N`
2. 너비와 높이 입력 (1-512 픽셀)
3. 배경색 선택 (흰색, 투명 등)

### 프로젝트 저장 (.plb)

1. **File → Save** 또는 `Ctrl+S`
2. 파일명 입력 (확장자 .plb 자동 추가)
3. 저장 내용:
   - 모든 픽셀 데이터
   - 색상 팔레트
   - 캔버스 크기
   - 메타데이터

### 프로젝트 불러오기

1. **File → Open** 또는 `Ctrl+O`
2. .plb 파일 선택
3. 이전 작업이 복원됩니다

**주의**: 저장하지 않은 작업은 손실될 수 있으니 먼저 저장하세요!

### PNG 내보내기

1. **File → Export → Export as PNG**
2. 파일명 입력
3. 내보내기 옵션:
   - **1x**: 픽셀 그대로 (32x32 → 32x32 PNG)
   - **2x**: 2배 크기 (32x32 → 64x64 PNG)
   - **4x**: 4배 크기
   - **8x**: 8배 크기
   - **사용자 지정**: 원하는 배율 입력

**용도**: 게임 스프라이트, 아이콘, 웹 이미지

### SVG 내보내기

1. **File → Export → Export as SVG**
2. 파일명 입력
3. 각 픽셀이 벡터 사각형으로 변환됩니다

**장점**:
- 무한 확대 가능
- PDF, 일러스트레이터 등에서 편집 가능
- 정확한 색상 유지

**용도**: 로고, 인쇄물, 벡터 그래픽 필요 시

## 편집 기능

### 실행 취소 / 다시 실행

- **실행 취소**: `Ctrl+Z` 또는 Edit → Undo
- **다시 실행**: `Ctrl+Y` 또는 Edit → Redo
- 무제한 히스토리 (메모리 허용 범위 내)

### 전체 선택 / 지우기

- **전체 선택**: `Ctrl+A`
- **전체 지우기**: `Delete` 또는 Edit → Clear Canvas
- **새 색으로 채우기**: Edit → Fill Canvas

## 팁과 요령

### 성능 최적화

- **큰 캔버스 (128x128 이상)**는 줌 아웃 시 느려질 수 있습니다
- 격자 끄기로 렌더링 속도 향상
- 작업 중 정기적으로 저장하세요

### 픽셀아트 기법

1. **제한된 팔레트 사용**: 8-16색 이내로 일관성 유지
2. **안티앨리어싱 피하기**: 날카로운 경계선 유지
3. **클러스터링**: 어두운 픽셀을 그룹으로 배치
4. **디더링**: 제한된 색상으로 그라데이션 표현

## 문제 해결

### 캔버스가 느려요
- 줌 레벨을 낮추세요
- 격자를 끄세요
- 캔버스 크기를 줄이세요

### 색상이 제대로 안 나와요
- 현재 선택된 색상을 확인하세요 (우측 상단)
- 투명도가 0이 아닌지 확인하세요

### 파일을 열 수 없어요
- .plb 파일인지 확인하세요
- 파일이 손상되지 않았는지 확인하세요
- 다른 버전의 PixeLab으로 저장된 파일일 수 있습니다
